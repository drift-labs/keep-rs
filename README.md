## JIT MAKER

1. create `.env` file with PRIVATE_KEY (as [u8]) and RPC_URL 
2. customize `JitMakerConfig` in `./jit/src/main.rs` 
3. run `bash jit.sh`

the author would generally advise to use an entirely separate account for jitting.
